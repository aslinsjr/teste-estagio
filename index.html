Questões sobre Desenvolvimento Web

Responda com suas palavras, não se preocupe com termos técnicos.

1. HTML, CSS e JavaScript

O que faz cada uma dessas tecnologias: HTML, CSS e JavaScript? Dê um exemplo de como usaria cada uma para criar uma página web simples.

2. Sites Responsivos

O que significa um site ser "responsivo"? Como você faria um site funcionar bem tanto no celular quanto no computador?

3. Git e Trabalho em Equipe

Explique o que é Git e para que serve. Como você organizaria seu código ao trabalhar com outros desenvolvedores?

4. Frameworks

Você conhece algum framework ou biblioteca (React, Bootstrap, etc.)? Se sim, explique o que é e para que serve. Se não, pesquise sobre um e conte o que entendeu.

5. Como Você Resolve Problemas

Conte sobre uma vez que você teve dificuldade para fazer algo funcionar no código. Como resolveu o problema? Onde buscou ajuda?
